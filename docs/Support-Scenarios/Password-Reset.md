# INC001 Password Reset

## Issue Description

User claims they forgot their password after a long vacation and needs an immediate reset. They say their manager already approved it and they just need IT to push the reset through.

## Troubleshooting Performed

- Verified user identity.
- Located account in Active Directory.
- Confirmed account was enabled.

## Resolution

- Reset user password.
- Enabled "User must change password at next logon."

## Verification

- User successfully authenticated.
- Password change completed.

## Key Takeaways

- Password expiration policies help maintain security
- Users on extended leave often face password expiration issues
- Temporary passwords should be communicated securely
- Users should change temporary passwords immediately upon login
